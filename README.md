# TomatoUI 自述
TomatoUI 是一个采用 Vue3+Vite2.x 构建的移动端组件库，依托最流行的框架技术，将巧妙的构思与精彩的设计完美结合形成一套通用组件，呈现给每一个开发者
# 愿望
做一款任何开发者都能够轻松，快速上手的移动端简约组件库，它不一定是最优的，最好的，但我们仍希望它可以帮助到你 :blush:
# 开发计划
我们的前期目标是实现功能，然后再逐步打磨组件的易用，简单！
